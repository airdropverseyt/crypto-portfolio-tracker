# 💰 Crypto Portfolio Tracker

A simple crypto portfolio tracker that lets you track your holdings and total value in real-time.

## 🚀 Features
- Add crypto holdings
- Live price fetching (CoinGecko API)
- Portfolio value calculation
- Local storage (data saved in browser)

## 🖥️ How to Run
1. Clone repo
2. Open index.html in browser

## ⚙️ Tech Stack
- HTML
- CSS
- JavaScript
- CoinGecko API
